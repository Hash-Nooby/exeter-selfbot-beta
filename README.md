beta release version for my discord selfbot

basically has everything you need for a selfbot plus more has a built in antinuke and music player too

## Notable Features

```python

- editsnipe
- antinuke
- musicplayer
- message logger
- mee6 auto level up
- nitro sniper
- giveaway sniper
- slotbot sniper
- image search
- image manipulation
- massreact
- copycat
- 24/7 yui kiss/hug
- many more
```
